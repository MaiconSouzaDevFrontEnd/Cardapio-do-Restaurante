3 - Filtrar pratos por categorias
Uma das funcionalidades da aplicação Cardápio de Restaurante é filtrar os pratos por categoria através dos botões.

Nesse caso você pode criar uma função de filtro (Array.filter)

Já com o Array.filter você poderá filtrar e buscar os dados da coleção.

Exemplo: Ao clicar no botão 'Massas', a aplicação irá filtrar e apresentar em tela os produtos da categoria Massas.

Figura 3. Filtragem por categoria
Você pode criar essa funcionalidade da seguinte forma:

Defina que os botões de categoria disparem uma função;
Essa função recebe como parâmetro a categoria;
A função filtra os pratos da coleção caso a categoria do prato seja igual ao parâmetro recebido.


Figura 4. Função de filtro por categoria
Veja no código abaixo o exemplo da função de filtro:

export const filtrarProdutos = (categoria) => {
  return produtos.filter((produto) => produto.categoria === categoria);
};

6 - Buscando por texto parcial
Para fazer uma busca por um trecho de um texto, ou mesmo buscar por um texto em qualquer posição de uma string utilizamos o método nativo do JavaScript, includes.

O includes verifica se um texto específico existe em uma string. Veja exemplos:


Figura 13. Includes sendo usado para encontrar palavras em uma string.
Desse modo, podemos utilizar o includes em um filter para criar buscas por textos digitados pelo usuário:

Figura 14. Filter com includes para fazer busca.
Veja nos códigos abaixo os exemplos apresentados:

//Exemplo1
const texto = "Halo é o principal game da plataforma xbox";
const contemTexto = texto.includes("xbox");

//Exemplo2
const texto = "A fazenda possui diversas plantações";
const contemTexto = texto.includes("planta");
Código 11: Exemplos de uso do includes

export const buscarProduto = (textoDigitado) => {
  return produtos.filter(
    (produto) =>
      produto.nome.toLowerCase().includes(textoDigitado.toLowerCase()) ||
      produto.descricao.toLowerCase().includes(textoDigitado.toLowerCase())
  );
};

7 - Separação do código em camadas
É importante para a organização do seu código que ele seja separado em camadas.

A camada visual da aplicação será responsável por exibir os componentes e os dados da aplicação.

Já a camada de serviço será responsável por conter a lógica da aplicação.

Na aplicação Cardápio de Restaurante a lógica se refere aos códigos responsáveis pelas filtragens dos dados e pela busca.
