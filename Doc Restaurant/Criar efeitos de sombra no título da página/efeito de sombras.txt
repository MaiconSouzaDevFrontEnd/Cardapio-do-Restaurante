Criar efeito de sombra no título da página
Você pode criar um efeito de sombra no título da página utilizando a propriedade CSS text-shadow.

Figura 1. Dica de como criar o sombreado do título.
Veja no código abaixo um exemplo de código CSS que cria um efeito de sombra utilizando a propriedade CSS text-shadow.

div h1 {
  letter-spacing: 5px;
  font-size: 46px;
  text-shadow: 0 -1px 4px #FFF, 0 -2px 10px #ff0, 0 -10px 20px #ff8000, 0 -18px 40px #F00;
}
